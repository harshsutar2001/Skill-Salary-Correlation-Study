# Skill-Salary Correlation Analysis
### Data for Career Intelligence

## Project Overview
Choosing the right skills can significantly impact career growth and salary potential.  
This project analyzes real-world job salary datasets to uncover relationships between **technical skills, experience levels, and salary outcomes**.

The objective is to identify **which skills deliver the highest return on investment in today's job market.**

---

## Key Questions Answered
- Which technical skills lead to the highest salaries?
- How does experience impact salary growth?
- Which skill combinations produce the best salary outcomes?
- Which roles offer the highest compensation across industries?

---

## Dataset Sources
Datasets used in this project were collected from:

- Kaggle: Data Science Salaries Dataset
- Job market salary reports
- Aggregated developer salary surveys

Dataset fields include:

- Job Title
- Skills
- Experience Level
- Salary
- Industry
- Country

## Project Workflow

### 1 Data Cleaning
- Standardized salary values into a single currency
- Removed duplicates and missing values
- Normalized skill names (Python3 → Python)
- Extracted skills into structured format

### 2 Feature Engineering
Created binary columns for top skills such as:
- Python
- SQL
- Excel
- Power BI


### 3 Exploratory Data Analysis
Performed analysis to identify:

- Average salary by skill
- Salary distribution by experience level
- Top paying job roles
- Skill combinations with highest salary impact

### 4 Data Visualization
Visualizations created include:

- Average Salary by Skill
- Salary Distribution
- Skill vs Experience Heatmap
- Top Paying Skill Clusters

### 5 Dashboard
An interactive **Power BI dashboard** was created to allow dynamic exploration of:

- Salary trends
- Skill demand
- Experience vs compensation

---

## Key Insights

Some major findings from the analysis:

- Python remains the highest-paying skill across multiple roles.
- Professionals with **Python + SQL + BI tools** earn significantly higher salaries.
- Salary increases strongly with experience after the 3–5 year mark.
- Cloud and machine learning related skills show strong salary premiums.
